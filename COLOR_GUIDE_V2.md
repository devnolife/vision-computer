## üìã Usage Guidelines

### Color Combinations (Recommended)

#### 1. **Fresh & Modern**
- Primary: `brand-blue` or `brand-aqua`
- Background: `brand-blue-light` or `brand-aqua-light`
- Text: `brand-navy-dark`
- Example: Feature cards, modern UI sections

#### 2. **Premium & Elegant**
- Primary: `brand-purple` or `brand-lavender`
- Background: `brand-purple-light` or `brand-lavender-light`
- Text: `brand-navy-dark`
- Example: Premium packages, special offers

#### 3. **Warm & Friendly**
- Primary: `brand-peach` or `brand-coral`
- Background: `brand-peach-light` or `brand-cream`
- Text: `brand-navy-dark`
- Example: Testimonials, welcoming sections

#### 4. **Clean & Professional**
- Primary: `brand-primary` (green)
- Background: `brand-mist` or `brand-sky`
- Text: `brand-navy-dark`
- Example: Business sections, professional content

#### 5. **Info & Communication**
- Primary: `brand-blue`
- Background: `brand-blue-light`
- Text: `brand-navy-dark`
- Example: Info boxes, announcements

## ‚úÖ Do's and ‚ùå Don'ts

### ‚úÖ Do's
- Use blue colors for informational content
- Use purple for premium features
- Use coral/peach for warm, friendly sections
- Mix and match different color families for variety
- Use light versions for backgrounds
- Use dark versions for text and emphasis

### ‚ùå Don'ts
- Don't use only green colors throughout the entire page
- Don't mix too many bright colors in one section
- Don't use coral for success states (use sage instead)
- Don't forget contrast for accessibility
- Don't overuse bright colors (use neutral backgrounds)

## üìä Color Distribution Recommendation

For a balanced, non-monotonous design:
- **30%** Primary (Green) - Main brand elements
- **20%** Blue - Informational elements
- **15%** Purple - Premium features
- **10%** Aqua/Teal - Fresh accents
- **10%** Warm (Peach/Coral) - Friendly touches
- **15%** Neutrals (Cream/Mist/Sky) - Backgrounds

This ensures variety while maintaining brand identity! üé®
